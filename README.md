# 灰度阅读实验

这个仓库包含 3 个静态网页实验，并在根目录提供统一入口页。

- `index.html`: 实验入口
- `display/`: 文本显示
- `adjustment/`: 灰度阅读调节实验
- `comparison/`: 灰度偏好比较实验

## GitHub Pages 发布

1. 在 GitHub 新建一个仓库。
2. 把本文件夹的全部内容上传到仓库根目录。
3. 打开仓库的 `Settings` -> `Pages`。
4. 在 `Build and deployment` 中选择：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. 保存后等待 GitHub Pages 生成访问链接。

发布后访问仓库的 Pages 链接，会先进入实验入口页，再选择进入 3 个实验之一。
